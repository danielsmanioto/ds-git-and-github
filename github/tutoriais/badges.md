# 🏆 GitHub Profile Achievements

A complete list of all currently available GitHub profile badges, how to earn them, and their tiers.

---

## ✅ Obtainable Achievements

### ⚡ Quickdraw

> Close a Pull Request or Issue within 5 minutes of it being opened.

- You can close your own issues — no external contribution needed.
- **Difficulty:** Very Easy

---

### 🤙 YOLO

> Merge a Pull Request without a code review.

- Requires at least one reviewer added, but merge without waiting for approval.
- **Difficulty:** Medium

---

### 🦈 Pull Shark

> Open Pull Requests that get merged.

- Can be done on your own repositories.
- **Difficulty:** Medium

| Tier | PRs Merged |
|------|-----------|
| Default | 2 |
| 🥉 Bronze | 16 |
| 🥈 Silver | 128 |
| 🥇 Gold | 1024 |

---

### 👥 Pair Extraordinaire

> Co-author commits in merged Pull Requests.

- Use `Co-authored-by: Name <email>` in commit messages, or use GitHub Desktop's co-author feature.
- **Difficulty:** Hard

| Tier | Co-authored PRs |
|------|----------------|
| Default | 1 |
| 🥉 Bronze | 10 |
| 🥈 Silver | 24 |
| 🥇 Gold | 48 |

---

### 🧠 Galaxy Brain

> Have answers accepted in GitHub Community Discussions.

- Participate at [github.com/orgs/community/discussions](https://github.com/orgs/community/discussions) and get your answers marked as accepted.
- **Difficulty:** Medium

| Tier | Accepted Answers |
|------|-----------------|
| Default | 2 |
| 🥉 Bronze | 8 |
| 🥈 Silver | 16 |
| 🥇 Gold | 32 |

---

### ⭐ Starstruck

> Have a repository receive a high number of stars.

- Focus on creating useful, well-documented open source projects and promoting them.
- **Difficulty:** Hard

| Tier | Stars |
|------|-------|
| Default | 16 ⭐ |
| 🥉 Bronze | 128 ⭐ |
| 🥈 Silver | 512 ⭐ |
| 🥇 Gold | 4096 ⭐ |

---

### 💖 Public Sponsor

> Sponsor an open source contributor, repository, or organization via GitHub Sponsors.

- Visit [github.com/sponsors/explore](https://github.com/sponsors/explore) and sponsor any project you like. Minimum is usually $1/month.
- **Difficulty:** Very Easy

---

## 🧪 In Testing (Not Yet Available)

### ❤️ Heart on Your Sleeve

> React to something on GitHub with a ❤️ emoji.

- **Status:** Beta — not yet earnable

---

### 🌐 Open Sourcerer

> Have Pull Requests merged in more than one public repository.

- **Status:** Beta — not yet earnable

---

## ⛔ No Longer Obtainable

### 🏔️ Arctic Code Vault Contributor

> Contributed code to repositories in the [2020 GitHub Archive Program](https://archiveprogram.github.com/).

- **Status:** Closed — program ended in 2020

---

### 🚁 Mars 2020 Contributor

> Contributed code to repositories used in NASA's [Mars 2020 Helicopter Mission](https://github.com/readme/nasa-ingenuity-helicopter).

- **Status:** Closed — mission-specific, no longer available

---

## 🎨 Skin Tone Variants

Some badges adapt to your emoji skin tone preference. You can change yours at [github.com/settings/appearance](https://github.com/settings/appearance).

Badges with skin tone variants:
- ⚡ Quickdraw
- ⭐ Starstruck
